# wdi-project1a-html-quiz-game
---
## A simple HTML5 quiz game

### Techonologies Used
- HTML 5
- CSS 3
- Vanilla Javascript
- jQuery

### Instructions
1. Choose player
2. Player will take turns to answer the quiz. 
3. Player with most correct answer wins.
4. Go to left panel to choose topics in Web Develoment.
